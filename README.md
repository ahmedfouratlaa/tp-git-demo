# TP Git & VS Code : Apprentissage Collaboratif

Bienvenue dans ce projet de démonstration ! 
L'objectif est de pratiquer les commandes de base de Git dans un environnement partagé.

## Instructions
1. Clonez ce dépôt.
2. Ajoutez votre nom dans `participants.md`.
3. Modifiez le style dans `index.html`.
4. Synchronisez vos changements !
